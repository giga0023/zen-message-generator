# Zen Message Generator

**Descrizione**
Questo piccolo progetto in Node.js stampa una frase zen casuale in console ogni volta che viene eseguito. È pensato come esercizio per imparare a lavorare con array, funzioni e Git, con la struttura pronta per essere ampliata in futuro.

**Funzionalità**

* Contiene un array di 50 frasi zen in italiano
* Funzione `randomPhrase(arr)` per estrarre una frase casuale
* Stampa automatica della frase in console all’avvio (`node main.js`)

**Come eseguire il progetto**

1. Clona la repository:

```bash
git clone https://github.com/giga0023/zen-message-generator.git
```

2. Spostati nella cartella del progetto:

```bash
cd zen-message-generator
```

3. Avvia lo script con Node.js:

```bash
node main.js
```
